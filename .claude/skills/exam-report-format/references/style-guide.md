# スタイルガイド：大学入試調査レポート

## 文体

- 敬体（です・ます調）を使用する
- 断定的な表現より「〜とみられる」「〜とされる」を使う（事実確認できない情報の場合）

## 数値の表記

- 配点：「○点満点中△点」のように満点を必ず併記する
- 倍率：「約○倍」（年度により変動する場合は年度を併記する）
- 募集人員：「約○名」（年度により変動する場合は年度を併記する）
- 年度：西暦（例：2027年度入試）で統一し、和暦のみの表記は避ける

## 情報の信頼性

- 大学公式サイト・入試要項・文部科学省など一次情報を優先する
- 一次情報で確認できなかった項目は「（情報なし）」と明記する（空白にしない）
- 出典によって数値に幅がある場合は、単一の断定値にせず幅・出典の違いを明記する
