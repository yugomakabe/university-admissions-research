## 概要

大学入試調査レポートのMarkdownを、5セクション構成のテンプレートに整形します。
沿革・設置学部一覧・キャンパス概要といった静的な「大学基本情報」は対象外（不要）。
ただし学部のキャンパス移転・改組・名称変更等の**変更のあった事実**は「入試制度の変更点」セクションで扱う。

対象セクション：

1. エグゼクティブサマリー
2. 入試方式の全体像
3. 学部・学科別 入試詳細
4. 入試制度の変更点（経年比較。科目・配点・選抜方式に加え、学部のキャンパス移転・改組・名称変更等の組織変更も含む）
5. 受験生・進路指導への示唆

## 手順

1. bash .claude/skills/exam-report-format/scripts/format-report.sh {入力ファイル} {出力ファイル} を実行して前処理を行う
2. templates/exam-report-template.md を読み込んでテンプレート構造を把握する
3. 入力ファイルのコンテンツを読み込んでセクションを識別する
4. 各セクションでは対象大学（N校）を横断比較できる形式（比較表・箇条書き）で整理する
5. 各コンテンツをテンプレートの対応セクションに配置する
6. 入力に存在しないセクション・大学には「（情報なし）」と記載する
7. 整形済みMarkdownファイルを指定の出力先に保存する

スタイルの詳細ルールは references/style-guide.md を参照すること。
